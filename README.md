# School Management Mini System

A simple full-stack web application built using the **MERN Stack** to manage students and their tasks/assignments.

## Tech Stack

Frontend:
- React
- Tailwind CSS
- React Router
- Axios
- React Toastify

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Features

### Authentication
- Admin Login
- JWT-based authentication
- Protected Dashboard routes

### Student Management
- Add Student
- View Students
- Update Student
- Delete Student

### Task / Assignment Management
- Assign tasks to students
- View all tasks
- Mark tasks as completed
- Delete tasks

### Dashboard
- Manage students and tasks in one place
- Clean UI using Tailwind CSS
